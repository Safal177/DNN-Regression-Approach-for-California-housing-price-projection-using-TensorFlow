# DNN-Regression-Approach-for-California-housing-price-projection-using-TensorFlow

# Project title:
California housing price estimation using Deep Neural Network (Regression) in TensorFlow

# Core Tech:
Basic tools are used:
•	Python                                                                                                                                                           
•	Scikit-learn                                                                                                                                                 
•	 NumPy 
•	 Pandas
•	TensorFlow |  Keras  
•	 Feature Scaling 
•	Deep Neural Networks (DNN)
•	Regression

# Project objective:
Key objectives are:
•	Using TensorFlow to develop a DNN regression model.
•	Standardizing features for model training
•	Training the model with separate training and validation sets
•	Using early stopping to decrease model overfitting.
•	Using mean absolute error to test regression accuracy
•	Applying a DNN pipeline for tabular data projection. 

# Core steps:
Key components are:
•	Dataset loading
•	Feature normalization
•	DNN structure
•	Model Initialization
•	Training the model
•	Training progress measuring
•	Evaluation of the model
